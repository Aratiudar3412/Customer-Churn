The goal of this project is to analyze customer data and build a machine learning model to predict customer churn. This helps businesses identify customers who are likely to leave and take actions to improve retention.
Dataset: Telco Customer Churn Dataset

Records: ~7,000 customers

Features include:

Demographics (Gender, Senior Citizen)

Account details (Tenure, Contract Type)

Services (Internet, Phone, Streaming)

Billing (Monthly Charges, Total Charges)

Target Variable: Churn (Yes/No)
